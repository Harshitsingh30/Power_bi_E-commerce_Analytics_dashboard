PROJECT OVERVIEW 

1. Project Title:
    
E-Commerce Sales & User Analytics Dashboard

2. Objective:
    
To analyze sales performance and user activity patterns in an e-commerce business. The 
insights generated from the analysis help support data-driven business decisions and improve 
overall performance.

3. Dataset Source: 
The dataset used for this project was sourced from Kaggle.
 
4. Tools Used:
    
● Microsoft Excel

● Power Query Editor

● MySQL

● Power BI 

5. Project Process :
    
(1) Excel: 

● Performed data summarization using Microsoft Excel.

● Cleaned and transformed data using Power Query Editor.

(2) MySQL:

● Created an E-Commerce database in MySQL.

● Imported CSV files into the database tables. 

● Performed data analysis using SQL queries. 

(3) Power BI: 

● Connected Power BI to the MySQL database.

● Created calculated columns and measures using DAX.

● Built an interactive dashboard for business analysis.

6. KEY INSIGHTS:

● The Pet Supplies category generates the highest number of orders, while the 
Electronics category generates the highest revenue.

● The Pet Supplies, Toys, and Clothing categories have the highest order quantities.

● Product views are high, but the purchase rate is comparatively low.

● Verdand Isles City has the highest number of users, whereas Krypton City has the 
lowest number of users. 

● Male users contribute a slightly higher percentage of total users compared to female 
users. 

7. RECOMMENDATIONS:
   
● Increase the purchase conversion rate by offering targeted discounts and promotional 
campaigns. 

● Maintain adequate inventory levels for high-demand categories such as Pet Supplies, 
Toys, and Clothing. 

● Reduce product return rates by improving product quality, descriptions, and delivery 
accuracy.
